## Development process
1. 安装Android studio

https://developer.android.com/codelabs/basic-android-kotlin-compose-install-android-studio

2. 配置环境变量
第 1 步：确认 SDK 路径（很关键，1 分钟）
你截图里 SDK 在：

`/Users/ponepuck/Library/Android/sdk`

打开终端（macOS 终端或 VS Code 终端都行）跑：

`/Users/ponepuck/Library/Android/sdk/platform-tools/adb version`

如果能输出版本号 → ✅ OK

再跑：

`echo $ANDROID_HOME`

如果是空的，我们就设置一下（VS Code 用起来更顺）：

`echo 'export ANDROID_HOME=/Users/ponepuck/Library/Android/sdk' >> ~/.zshrc`

`echo 'export PATH=$PATH:$ANDROID_HOME/platform-tools' >> ~/.zshrc`

`source ~/.zshrc`

`adb version`
