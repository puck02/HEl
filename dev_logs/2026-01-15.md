## Overview
- AI 建议 prompt 现包含 30 天窗口（作为背景），仍以今日 + 近 7 天为主信号。
- 首页/日报的随机关怀语提升到 App 级 rememberSaveable，每次启动固定，不会因切 Tab 变化。
- 设置页改为软卡片/渐变按钮风格，含 AI 开关卡片、可显隐的 API Key 输入、提示卡与导出/导入按钮。

## Development process
1. 更新 AdvicePromptBuilder：拼接 30 天指标块，提示模型以今日+7 天为主、30 天为辅。
2. 在 MainActivity 生成并下发 home/daily care prompts，DailyHeader 接收外部文案；移除局部随机函数，新建 core/ui/CarePrompts。
3. 重绘 SettingsScreen：卡片化 AI 开关与 API Key 输入，添加显隐切换、保存/清除行内按钮、信息提示卡、渐变导出与描边导入按钮。
4. 编译验证 `./gradlew :app:compileDebugKotlin`，通过但存在既有警告（MainActivity always-true 检查、DailyReportViewModel 重名 label、OutlinedTextField 颜色 API 过时提醒）。
